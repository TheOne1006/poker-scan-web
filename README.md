# proker-san web

这是一个用于 app 嵌入的 PWA 应用, 这里主要包含 三个页面

- 需要界面 UI 接近原生 APP UI
- 支持theme: default、dark theme
- 需要 ios，也需要 android 的样式 

## mock serve

- 启动一个 mock server 模拟 后台 api 与 返回

## 隐私政策页面 `Privacy`

- 解析 markdown 文件 构建对应的 html

## 服务条款页面 `Terms`

- 解析 markdown 文件 构建对应的 html

## 智能客服聊天页面 `Support`

- 类似聊天对话，支持图片发送 提问
- 根据 api 返回 信息
